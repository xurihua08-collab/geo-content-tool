# GEO 工具升级规划 - 批量生成 + 自动分发

## 当前功能
- [x] 内容分析（清晰度、EEAT、引用友好度等）

## 计划新增功能

### 1. 内容批量生成模块
- 基于关键词/主题自动生成 GEO 优化内容
- 支持多种内容类型（文章、问答、产品描述等）
- 内置 EEAT 优化模板
- 批量生成多篇文章

### 2. 自动分发模块
- 对接 CMS 系统（WordPress、Shopify 等）
- 社交媒体分发（微信公众号、知乎、小红书等）
- 邮件/消息推送
- 定时发布功能

### 3. 工作流自动化
- 生成 → 分析 → 优化 → 分发 全流程
- 批量任务队列
- 执行日志和报告

## 技术架构
```
geo_platform/
├── analyzer/          # 现有分析模块
├── generator/         # 内容生成模块
├── publisher/         # 分发模块
├── scheduler/         # 定时任务
├── dashboard/         # 管理后台
└── api/              # API 接口
```

## 实现步骤
1. 先实现内容生成模块（调用 LLM API）
2. 添加分发接口（从简单平台开始）
3. 整合工作流
4. 添加管理界面
