# 内容分发管理系统

🚀 一个支持多品牌管理、多平台账号绑定、文章发布追踪的内容管理系统。

## 功能特性

### 1. 多品牌管理
- ✅ 支持添加多个品牌
- ✅ 每个品牌独立配置
- ✅ 品牌信息包括：名称、描述、官网、行业

### 2. 平台账号管理
- ✅ 支持 5 个平台：Google SEO、知乎、今日头条、百家号、福步论坛
- ✅ 每个品牌可绑定多个平台账号
- ✅ 账号信息存储：账号名、账号ID、API Key、Cookies
- ⚠️ 暂不支持账号登录验证（需手动确认账号有效）

### 3. 文章管理
- ✅ 按品牌管理文章
- ✅ 文章状态追踪：草稿、发布中、已发布、失败
- ✅ 支持关键词标签

### 4. 发布管理
- ✅ 发布记录列表
- ✅ 状态筛选：成功、失败、发布中
- ✅ 发布链接追踪
- ✅ 错误信息记录

## 技术栈

- Python 3.8+
- Streamlit 1.28+
- JSON 文件存储（无需数据库）

## 安装运行

```bash
# 安装依赖
pip install -r requirements.txt

# 启动应用
streamlit run app.py
```

## 使用流程

1. **创建品牌** → 在「品牌管理」页面添加品牌
2. **绑定账号** → 在「平台账号」页面为品牌绑定各平台账号
3. **创建文章** → 在「文章管理」页面撰写文章
4. **发布追踪** → 在「发布管理」页面查看发布状态

## 数据存储

所有数据存储在 `data/` 目录下的 JSON 文件中：
- `brands.json` - 品牌数据
- `articles.json` - 文章数据
- `publish_records.json` - 发布记录

## 注意事项

1. **账号验证**：系统不验证平台账号是否可登录，请确保填写的账号信息正确
2. **发布功能**：当前版本仅记录发布状态，实际发布需在各平台手动操作
3. **数据备份**：定期备份 `data/` 目录以防数据丢失
